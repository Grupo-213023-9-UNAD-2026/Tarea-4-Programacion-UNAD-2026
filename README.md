## 📋 Descripción

Sistema desarrollado en Python que gestiona **clientes, servicios y reservas** para la empresa ficticia Software FJ, sin uso de bases de datos. Todo el estado del sistema se mantiene en memoria mediante objetos y listas. Los errores se registran en un archivo `sistema.log`.
